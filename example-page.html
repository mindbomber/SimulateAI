Example Page

This page demonstrates how to use the SharedNavigation component.

How it works:

  1. Add the page-id meta tag to identify the current page
  2. Include the shared-navigation.css stylesheet
  3. Include the shared-navigation.js script
  4. The navigation will be automatically loaded and injected

Features:

  • Organized navigation groups (Primary, Community, Actions, Auth)
  • Mega menu for categories with search functionality
  • Dropdown menus for Community and About sections
  • Mobile-responsive with hamburger menu
  • Accessibility features (keyboard navigation, ARIA attributes)
  • Automatic active page detection and highlighting

Implementation Guide

HTML Structure:

<div class="breadcrumb-container">
    <div class="breadcrumb-wrapper">
        <nav aria-label="Breadcrumb">
            <ol class="breadcrumb">
                <li class="breadcrumb-item">
                    <a href="#home">
                        <span class="breadcrumb-icon breadcrumb-home">üè†</span>
                        Home
                    </a>
                </li>
                <li class="breadcrumb-item">
                    <a href="#page">Page</a>
                </li>
                <li class="breadcrumb-item active">Current Page</li>
            </ol>
        </nav>
    </div>
</div>

Key Features:

  • Responsive Design: Automatically adapts to mobile screens
  • Accessibility: Proper ARIA labels and semantic HTML
  • Dark Mode Support: Seamless integration with your theme system
  • Glass Morphism: Modern backdrop blur effects
  • Interactive Elements: Action buttons and search integration
  • Customizable: Easy to modify colors and spacing

CSS Classes:

  • .breadcrumb-compact - Collapsed view for mobile
  • .breadcrumb-item.active - Current page styling
  • .breadcrumb-actions - Action buttons container
  • .breadcrumb-search - Integrated search field
